# 🎨 LEXIS English — Design System

## 1. Phong cách & Chủ đạo
- **Style:** Bold & Playful — màu sắc tươi sáng, góc bo tròn, trẻ trung, năng động.
- **Theme:** Hỗ trợ Dark mode + Light mode (có toggle chuyển đổi).

## 2. Typography
- **Font chính (Display/Headings):** Nunito (tròn trịa, thân thiện).
- **Font phụ (Mono/IPA):** Space Mono (dùng cho phiên âm IPA, code, hoặc các thông số kỹ thuật).

## 3. Bảng Màu (Color Palette)

| Tên | Hex | Chức năng / Dùng cho |
|---|---|---|
| Green | `#00D68F` | Primary action (nút chính), hiển thị streak, trạng thái "Mastered" (đã thuộc). |

| Purple | `#8B5CF6` | Module Grammar (Ngữ pháp), badge hiển thị level. |
| Blue | `#3B82F6` | Module Listening (Luyện nghe), các icon/text mang tính chất thông tin (info). |

| Pink | `#F472B6` | Hiển thị Badges (Thành tựu). |
| Dark BG | `#0F0F14` | Màu nền (background) cho Dark mode. |

| Light BG | `#F4F2FF` | Màu nền (background) cho Light mode. |
